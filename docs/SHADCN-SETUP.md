# shadcn/ui Components Installation Guide

## Components to Install

Run these commands one by one or all at once:

```bash
# Essential UI components
pnpm dlx shadcn@latest add button
pnpm dlx shadcn@latest add card
pnpm dlx shadcn@latest add badge
pnpm dlx shadcn@latest add input
pnpm dlx shadcn@latest add label
pnpm dlx shadcn@latest add progress
pnpm dlx shadcn@latest add separator
pnpm dlx shadcn@latest add tabs
pnpm dlx shadcn@latest add dialog
pnpm dlx shadcn@latest add dropdown-menu
pnpm dlx shadcn@latest add select
pnpm dlx shadcn@latest add slider
pnpm dlx shadcn@latest add toast
pnpm dlx shadcn@latest add alert
```

## Or Install All at Once

```bash
pnpm dlx shadcn@latest add button card badge input label progress separator tabs dialog dropdown-menu select slider toast alert
```

## What Each Component Is For

- **button**: Action buttons throughout the app
- **card**: Tool cards, containers
- **badge**: Status badges, tags
- **input**: File inputs, text fields
- **label**: Form labels
- **progress**: Upload/processing progress bars
- **separator**: Visual dividers
- **tabs**: Tool options, settings
- **dialog**: Modals, confirmations
- **dropdown-menu**: Navigation menus
- **select**: Dropdown selections (format, quality)
- **slider**: Quality/compression sliders
- **toast**: Success/error notifications
- **alert**: Info messages, warnings

## After Installation

The components will be added to `components/ui/` with proper dark theme styling.
